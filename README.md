# Python Basics — Автотесты калькулятора

Проект для изучения автоматизированного тестирования на Python.

## Что внутри:
- `Calculator.py` — простой калькулятор (сложение, вычитание, умножение, деление)
- `test_calculator.py` — базовые автотесты
- `test_calculator_parametrize.py` — параметризованные тесты (20 проверок)

## Как запустить тесты:
```bash
pytest test_calculator_parametrize.py -v
